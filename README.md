# CryptoWallet MVP

Кроссплатформенное мобильное приложение криптокошелька на React Native с поддержкой мультивалютного холодного хранения.

## Основные возможности

### MVP (Текущая версия)
- Просмотр баланса
- Пополнение через OKX
- Перевод на другие кошельки
- Мультивалютное отображение
- Холодное хранение ключей

### Планируемые функции
- Новостной модуль
- Маркетплейс
- Расширенная аналитика

## Разработка

### Требования
- Node.js 20+
- JDK 17
- Android Studio
- Android SDK

### Технологии
- React Native
- TypeScript
- Trust Wallet Core SDK

## Установка

```bash
# Установка зависимостей
npm install

# Запуск Android
npm run android
```
