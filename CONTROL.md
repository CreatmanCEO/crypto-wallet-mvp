# CONTROL LOG - CryptoWallet MVP

## 📱 Проект: Кроссплатформенный Криптокошелек
**Текущий статус:** Инициализация проекта
**Последнее обновление:** 20.02.2025

### 🎯 Цели проекта
1. Создание мобильного криптокошелька с поддержкой основной криптовалюты
2. Интеграция с OKX для операций
3. Поддержка мультивалютного отображения
4. Безопасное хранение ключей

### 🔧 Текущая структура проекта
```
src/
├── app/                    # Конфигурация приложения
│   ├── store/             # Redux store
│   └── navigation/        # Навигация
├── core/                  # Ядро приложения
│   ├── wallet/           # Интеграция с Trust Wallet Core
│   ├── blockchain/       # Web3 взаимодействие
│   └── security/        # Криптография и безопасность
├── features/             # Основные функции
│   ├── auth/            # Аутентификация
│   ├── wallet/          # Операции с кошельком
│   └── settings/        # Настройки приложения
├── shared/              # Общие компоненты
│   ├── components/      # UI компоненты
│   ├── hooks/          # Кастомные хуки
│   ├── constants/      # Константы
│   └── utils/          # Утилиты
└── services/           # Внешние сервисы
    ├── api/           # API клиенты
    └── storage/       # Локальное хранение
```
